# CloudBuddy ☁️

**CloudBuddy** — это аналог тамагочи с искусственным интеллектом. Ваш питомец — облачко, которое живёт в приложении, общается через чат (OpenAI) и иногда скучает. Когда ему скучно, с ним нужно поиграть или «выгулять»: пройти мини-игру в стиле Flappy Bird, летая между облаками.

## Идея

- **Облачко-друг** — виртуальный питомец с характером и настроением.
- **Чат с ИИ (OpenAI)** — вы общаетесь с облачком, оно отвечает умно и по-разному в зависимости от контекста.
- **Скука и забота** — если долго не заходить или не играть, облачку станет скучно; нужно его развлекать.
- **«Выгул» = мини-игра** — игра, похожая на Flappy Bird: вы управляете облачком (или птичкой), пролетаете между препятствиями-облаками, набираете очки. Так вы «выгуливаете» питомца и поднимаете ему настроение.

## Текущее состояние

- ✅ Android-приложение (Kotlin, Jetpack Compose)
- ✅ Мини-игра в стиле Flappy Bird (Canvas, тап = прыжок, препятствия-облака, счёт, Game Over)
- ⏳ Интеграция OpenAI для чата — в планах
- ⏳ Экран питомца, настроение, скука — в планах
- ⏳ Связь между игрой и «настроением» облачка — в планах

## Технологии

- **Kotlin**, **Jetpack Compose**, **Material 3**
- **Min SDK 26**, **Target SDK 34**, **Compile SDK 35**
- Сборка: Gradle (Kotlin DSL), версии в `gradle/libs.versions.toml`

## Как запустить

1. Клонировать репозиторий.
2. Открыть проект в Android Studio.
3. Подключить устройство или эмулятор, нажать Run.
